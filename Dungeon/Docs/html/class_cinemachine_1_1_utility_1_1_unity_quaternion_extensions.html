Get the rotations, first about world up, then about (travelling) local right, necessary to align the quaternion's forward with the target direction.

This represents the tripod head movement needed to look at the target. This formulation makes it easy to interpolate without introducing spurious roll.

Parameters

lookAtDirThe worldspace target direction in which we want to look
worldUpWhich way is up
