static string Cinemachine.Editor.SerializedPropertyHelper.PropertyName ( Expression< Func< object >>  exp)
static

This is a way to get a field name string in such a manner that the compiler will generate errors for invalid fields.

Much better than directly using strings. Usage: instead of

"m_MyField";

do this:

MyClass myclass = null; SerializedPropertyHelper.PropertyName( () => myClass.m_MyField);
